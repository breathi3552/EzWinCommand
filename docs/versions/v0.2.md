# EzWinCommand v0.2

本文件只记录轻量产品变化；详细当前行为以 `docs/product/` 为准。

## 当前能力摘要

- Windows Server、Web 管理/控制端和 Android App 局域网协同。
- 配对鉴权、设备管理、插件命令、异步长任务。
- Android 媒体卡、播放控制、音量和默认音频设备控制。
- 媒体 SSE 状态同步、局域网发现和 Windows 生命周期集成。

## 验证状态

- Server 和 Android 已有单元、接口及 Robolectric 覆盖。
- Android 模拟器 + 真实 Server UI/E2E 框架待建设。
- 真实 Windows Media/Core Audio、UAC/防火墙、真机网络、视觉和 TalkBack 存在待人工验证项。
