# REQ-003: 防火墙与自启动

- **日期**: 2026-07-02
- **版本**: v0.1
- **状态**: 已完成

## 原始需求

实现以下功能：
1. 防火墙自动配置：Server 启动时自动添加 Windows 防火墙入站规则，确保局域网设备可以访问 EzWinCommand 服务
2. 开机自启动：通过注册表 Run 键注册自启动，使用 pythonw.exe 实现无控制台窗口运行
3. CLI 参数集成：通过 `--install` / `--uninstall` 参数管理自启动注册
4. 无窗口启动：提供 start-hidden.bat 入口脚本，确保启动时不弹出控制台窗口
