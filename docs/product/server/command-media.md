# Server：命令与媒体

## 职责

发现并执行插件命令，为长任务提供异步状态，并对外提供 Windows 媒体与音频状态和控制。

## 特性与功能点

- `CMD-01`：发现插件并返回可用动作。
- `CMD-02`：短命令同步执行并返回清晰错误。
- `CMD-03`：长任务异步受理，客户端可轮询状态。
- `MEDIA-01`：提供当前媒体元数据、播放状态、封面和 revision。
- `MEDIA-02`：上一首、播放/暂停、下一首控制。
- `AUDIO-01`：主音量和默认输入/输出设备状态及控制。
- `MEDIA-03`：通过 SSE 推送单调 revision；慢消费者采用 latest-wins，必要时可回放。

## 关键约束

- 用户可见错误保持简短；服务端内部异常记录脱敏诊断和 traceback。
- 没有活动媒体时，音量和设备能力仍可用。
- 真实播放器和 Core Audio 结果需要 Windows 环境证据。

## 主要测试

参见 `TC-CMD-*`、`TC-MEDIA-*`、`TC-AUDIO-*` 和 `TC-XPLAT-*`。
